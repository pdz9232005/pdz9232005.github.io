# 图片颜色可视化网页

一个基于前端实现的图片颜色聚类与可视化网页。用户可以上传本地图片，对图片像素颜色进行 K-means 聚类分析，并通过 ECharts 以饼图或柱状图形式展示结果。同时支持 RGB / LAB 颜色空间切换，并提供基于大语言模型的颜色和谐度分析功能。

## 项目简介

本项目用于实现课程作业“图片颜色的可视化网页”。主要目标是对图片中的像素颜色进行聚类，并将聚类结果用图表进行展示。

## 功能特点

### 1. 图片上传与预览
- 支持上传本地图片文件
- 页面中实时显示图片预览

### 2. K-means 颜色聚类
- 对图片像素颜色进行聚类分析
- 支持交互式选择聚类数 K
- 当前页面中 K 值范围为 2 到 15

### 3. 颜色空间切换
- 支持 **RGB**
- 支持 **LAB**

### 4. 图表可视化
- 支持 **饼图**
- 支持 **柱状图**
- 使用 **ECharts** 进行可视化展示
- 图表中每个颜色区域直接使用对应聚类中心颜色进行显示
- 图表中可展示颜色编号、颜色值、像素数量和占比信息

### 5. AI 颜色和谐度分析
- 可对聚类得到的颜色组合进行和谐度分析
- 支持输入 API 密钥调用大语言模型接口
- 当接口不可用时，提供本地分析兜底逻辑

- 项目已成功部署到 GitHub Pages
