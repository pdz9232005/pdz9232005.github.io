# 图片颜色可视化网页

本项目是一个基于前端实现的图片颜色聚类与可视化网页。用户可以上传本地图片，对图片中的像素颜色进行 K-means 聚类分析，并使用 ECharts 以饼图或柱状图的形式展示聚类结果。项目同时支持 RGB / LAB 颜色空间切换，并提供颜色和谐度分析功能。

## 项目功能

### 1. 图片上传与预览
- 支持从本地选择图片文件
- 上传后可在页面中查看图片预览
- 未上传图片时页面会显示占位提示信息

### 2. 颜色聚类分析
- 使用 K-means 算法对图片像素颜色进行聚类
- 支持用户通过滑块调整聚类数量 K
- 当前页面中 K 值可设置范围为 2～15
- 根据课程要求，实际使用时建议设置 **K ≥ 5**

### 3. 颜色空间切换
- 支持 **RGB** 颜色空间
- 支持 **LAB** 颜色空间
- 在提取像素后，可根据用户选择决定是否将 RGB 转换为 LAB 再进行聚类

### 4. 图表可视化
- 使用 **ECharts** 对聚类结果进行展示
- 支持两种图表类型：
  - 饼图：展示不同颜色类别的占比
  - 柱状图：展示不同颜色类别的像素数量
- 图表中的每个区域颜色直接使用对应聚类中心颜色进行显示
- 支持鼠标悬停查看颜色编号、十六进制颜色值、像素数量、比例等信息

### 5. 颜色和谐度分析
- 支持对聚类得到的颜色组合进行和谐度分析
- 页面提供 API 密钥输入与保存功能
- 可尝试调用大语言模型接口进行分析
- 当接口不可用时，程序会自动回退到本地分析逻辑

## 页面说明

### 左侧：分析设置
- 选择图片
- 调整聚类 K 值
- 选择颜色空间（RGB / LAB）
- 选择图表类型（饼图 / 柱状图）
- 点击“开始分析”执行聚类
- 点击“检查和谐度”查看颜色搭配分析结果
- 配置并保存 API 密钥

### 右上：图片预览
- 显示当前选择的图片
- 若未选择图片，则显示占位提示

### 右下：分析结果
- 显示 ECharts 图表
- 显示颜色和谐度分析结果
- 在尚未分析前显示默认提示信息

## 核心实现说明

### 1. K-means 聚类
项目在前端使用 JavaScript 实现 K-means 聚类，主要流程包括：
1. 随机选取初始聚类中心
2. 计算每个像素到各中心的距离
3. 为像素分配最近类别
4. 重新计算每一类的均值中心
5. 经过多轮迭代后输出最终结果

### 2. 颜色空间转换
程序提供 `rgbToLab` 函数，用于将 RGB 颜色转换为 LAB 颜色空间，从而支持两种颜色空间下的聚类分析。

### 3. 图表绘制
程序会先统计各聚类中的像素数量，再将聚类中心颜色转换为十六进制颜色值，最终生成饼图或柱状图所需的数据结构。

### 4. 和谐度分析
程序优先尝试调用大语言模型接口；若接口不可用，则会采用本地生成的默认分析结果进行展示，以保证功能可正常演示。

### 本地运行
1. 将以下文件放在同一目录下：

   - `main.js`
   - `echarts.min.js`
2. 使用浏览器打开 `index.html`
3. 上传图片并开始分析
